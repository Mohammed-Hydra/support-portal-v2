# Credentials Checklist — Support Portal v2

**Do not write real passwords, API keys, or secrets in this file or anywhere in the repo.**
Store real values only in: Vercel → Project → Settings → Environment Variables, or local .env (never commit .env).

Use this checklist to track which credentials you need and where to set them.

--- 

1. DATABASE (PostgreSQL)
   Credential: DATABASE_URL — Full connection string from Neon or Supabase.
   Where to get: Neon: neon.tech → project → Connection string. Supabase: supabase.com → Project → Settings → Database.
   Where to set: Vercel env vars, or local .env
   PGSSLMODE: optional (require for cloud, disable for local).

2. APP SECRETS (required)
   JWT_SECRET — Long random string for login tokens.
   REQUESTER_SESSION_SECRET — For requester magic-link sessions.
   INGEST_HOOK_TOKEN — Optional; for webhook auth.
   Where to set: Vercel env vars, local .env

3. FIRST ADMIN USER
   SUPPORT_ADMIN_EMAIL, SUPPORT_ADMIN_PASSWORD, USER_EMAIL_DOMAIN
   Where to set: Vercel env vars, local .env

4. PORTAL URLS (for links in emails)
   PORTAL_BASE_URL — e.g. https://support.hydra-tech.pro
   REQUESTER_PORTAL_BASE_URL — e.g. https://support.hydra-tech.pro/public/requester/portal
   Where to set: Vercel env vars, local .env

5. EMAIL — RESEND (recommended)
   RESEND_API_KEY — from resend.com → API Keys → Create
   RESEND_FROM — Sender address (verified domain in Resend)
   Login: resend.com (account not stored in app)

6. EMAIL — MICROSOFT 365 (Graph)
   M365_TENANT_ID, M365_CLIENT_ID, M365_CLIENT_SECRET, M365_SENDER_UPN
   Where to get: Azure Portal → App registrations → your app
   Login: portal.azure.com (account not stored in app)

7. EMAIL — SMTP (fallback)
   SMTP_HOST, SMTP_PORT, SMTP_USER, SMTP_PASS, SMTP_FROM, SMTP_SECURE
   Where to get: Your email provider (e.g. smtp.office365.com)

8. VERCEL (hosting)
   Set all env vars at vercel.com → your project → Settings → Environment Variables.
   Login: vercel.com (for dashboard only)

9. GITHUB (code repo)
   github.com — push/pull code; connect Vercel. Not stored in app.

10. CURSOR (editor)
    cursor.com — IDE for editing project. Not used by running app.
